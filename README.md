# bc-vo1-payments ![test-and-deploy](https://github.com/gowda/bc-vo1-payments/workflows/test-and-deploy/badge.svg)


RESTful HTTP API for `payments` resource

## Run
### Setup
```bash
$ bundle install
$ bin/setup
```

### Server
```bash
$ bin/rails server
```

### Tests
```bash
$ bin/rails spec

# Or
$ bundle exec rspec
```

### Rubocop
```bash
$ bin/rails cop

# Or
$ bundle exec rubocop
```
